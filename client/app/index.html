YouTube Video
Editor Utility.

Cut, loop and re-publish any YouTube Video instantly.

  • No uploading or downloading video files
  • Precision loop timing (50ms resolution)
  • Whitelisted to play in the Twitter feed
  • Clean, mobile ready embeds
  • Free, online, open source.

    •     START EDITING     Only on Tablet and Desktop

    Editor Features

      Search

    Search YouTube's video and channels database with filters for duration, date and relevance.

      Player

    Cyclops is just a light-weight wrapper around the YouTube player that uses YouTube's own javascript API. This means your video edit gets served with the same speed and reliability as any YouTube video.

      Seek

    Jump to any location in the video by clicking the player timebar. Drag the rangefinder's handles to 'scrub' through the video, looking for the scene you want.

      Set

    Use the startpoint and endpoint setters to make micro adjustments to the beginning and end of your clip. Set clip boundaries as the video plays using the quickset buttons.

      Playback

    Set your clip's playback speed and sound level.

      Publish

    Generate a link or an iframe snippet for your clip and post or embed it in your page. See the Twitter guide below for help with links, and the Tumblr guide for help with iframes.

      Tumblr Guide

      Embed

    Click the 'Embed' button to get an iframe code snippet. By default Cyclops generates code for a responsive player, but you can also set the player size by selecting the 'Fixed' option and specifying its pixel height and width.

      Post

    Copy the iframe snippet and open a Tumblr share dialog by clicking the editor's share button. Press ctrl-v to paste the iframe code in the text body of your post. A thumbnail of your clip will appear in the Tumblr feed and the video edit itself will appear on your blog page.
